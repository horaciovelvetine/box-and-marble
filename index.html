The Problem

You are given 3 matchboxes by a professor with a conspicous grin. Each box contains exactly 2 marbles which are either red or white. The boxes are each incorrectly labeled as follows:

  • "Red and White"
  • "Red and Red"
  • "White and White"

The labels correctly outline the only possible combinations of marble colors in the boxes, but are currently incorrect.

Challenge: You can peek inside ONE box and look at exactly ONE marble. Can you figure out what's in each box?
